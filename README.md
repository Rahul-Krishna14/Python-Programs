### SIMPLE CALCULATOR

### OPRATIONS PERFORMED:
* Addition 
* subtration
* multiplication
* division

#### IMPLEMENTATION
```python
class Calculator:

    def __init__(self, a, b):
        self.a = float(a)
        self.b = float(b)

    def addition(self):
        return self.a + self.b

    def subtraction(self):
        return self.a - self.b

    def multiplication(self):
        return self.a * self.b

    def division(self):
        return self.a / self.b
```
